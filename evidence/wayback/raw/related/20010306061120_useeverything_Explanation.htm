What's a WAP browser?

A WAP browser is like a web browser, except it's designed to work on a small device such as a mobile phone. Rather than displaying normal web pages, it uses specially-prepared WAP pages. These are very similar to web pages, except they're smaller, and don't use a lot of graphics.

Many of the mobile phones that are now being sold (in Europe at least) have WAP browsers.

 

What's a text adventure?

A text adventure is a computerized interactive story. Like a story, everything is presented as text- there are no graphics. You get to choose what to do next by entering commands. These are usually fairly simple, such as "Take the wine glass", or "Attack the wolves with the fruit salad".

Traditionally you type in what you want to do, and the computer tries to make sense of it, sometimes not completely successfully. The adventure game at UseEverything.com doesn't bother with all this, as it would be too frustrating to type every command using the telephone keypad. Instead, it uses a "point and click" interface.

Anyway, the best way to learn about all this is to give it a go.
