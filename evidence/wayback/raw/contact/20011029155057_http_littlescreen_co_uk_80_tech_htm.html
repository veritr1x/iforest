Technical Details
(only read this if you're really interested in this sort of thing!)





The games on this site are implemented as Java servlets. Each time you make a move on either game, a servlet gets run on the littlescreen.co.uk server, and it provides the response to your move.

Beyond that, each game works quite differently, even though they may appear similar while playing them.
